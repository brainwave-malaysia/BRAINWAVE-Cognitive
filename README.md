## What is Inside

A single self-contained HTML file with 5 fully interactive wellbeing tools grounded in cognitive neuroscience.

| Tool | What it does | Neuroscience |
|---|---|---|
| Box Breathing | 4-count breathing timer with animated circle | Activates vagus nerve, shifts nervous system to safety |
| Stroop Test | 12-trial colour-word inhibition test with scoring | Measures prefrontal override of automatic responses |
| CBT Thought Record | 4-step guided cognitive reframe | Reappraisal recruits prefrontal cortex to modulate amygdala |
| PMR Sequence | Step-by-step progressive muscle relaxation | Discharges stored stress, activates parasympathetic system |
| Tapping EFT | 10-point acupoint tapping sequence | Somatic signalling calms amygdala activation |
